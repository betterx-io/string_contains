highLightWords property

List<String> highLightWords
final

highLightWords is the List of Strings to be used for the Text widget. This can be used to highlight certain words in the text. The words will be highlighted with the highlightWordsStyle. if highLightWords is provided, and not added StringContainsElementType.words in types, then StringContainsElementType.words will be added to types automatically.
